## Public Free GPU Platforms

> Long-term or renewable free GPU access for learning and experimentation.

- **Kaggle Notebooks**  
  Weekly renewable GPU/TPU quotas (T4 / P100).  
  Ideal for coursework, experiments, and benchmarks.

- **Baidu PaddlePaddle AI Studio**  
  Daily compute points, supports GPU training.  
  Stable access in mainland China.

- **Tencent Cloud Studio (GPU Workspace)**  
  Monthly free GPU minutes (~10,000 minutes).  
  Browser-based development environment.

## Trial & Limited-Time GPU

> Useful for demos, short experiments, or quick prototyping.

- **Google Colab (Free Tier)**  
  GPU/TPU available, but unstable for long jobs.

- **AWS SageMaker Studio Lab**  
  Limited daily GPU access for learning and testing.

- **Huawei CodeLab / Alibaba Tianchi GPU**  
  Occasional free GPU quotas via events or programs.

## Practical Tips

- Combine **Kaggle + Colab** for daily experiments
- Use **public platforms** for small or medium models
- Prefer **grant-based programs** for long-running or multi-GPU training
- Stability matters more than peak GPU specs for real research
